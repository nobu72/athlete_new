# athlete
このアプリはスポーツを行うアスリートへ向けて開発しました。
日々の練習や試合の動画を選手とコーチで共有し、レヴューが出来ます。
時間の無い中、ミーティングで集まれなくても自由にコミニュケーションが取れます。
